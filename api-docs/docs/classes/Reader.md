# Class: Reader

**`classdesc`** Reader reads messages from Kafka.

**`example`**

```javascript
// In init context
const reader = new Reader({
  brokers: ["localhost:9092"],
  topic: "my-topic",
});

// In VU code (default function)
const messages = reader.consume({ limit: 10 });

// In teardown function
reader.close();
```

# API Docs

Use the versioned references in this directory to match the API surface you are
building against.

- Latest v2 declaration file: [v2/index.d.ts](./v2/index.d.ts)
- Latest v2 generated reference: [v2/docs/README.md](./v2/docs/README.md)
- Legacy unversioned reference snapshot: [docs/README.md](./docs/README.md)

The v2 reference includes the deprecated `Writer`, `Reader`, and `Connection`
aliases together with their `Producer`, `Consumer`, and `AdminClient`
migration targets.
